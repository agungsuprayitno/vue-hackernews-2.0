# Indomog Voucher Generator Frontend

Frontend for Indomog Voucher Generator Using VueJs with Server Side Render.

## Getting Started

**Requires Node.js 8+ and Vue 2 + SSR**

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at localhost:8080
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
